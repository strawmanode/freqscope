# Screenshots

Images referenced by the main [README](../README.md). Drop PNGs here with these
exact names and they'll render automatically:

| File          | What to capture                                                       |
|---------------|----------------------------------------------------------------------|
| `scope.png`   | **Hero** — the full radar scope page with live traffic (e.g. KATL).  |
| `search.png`  | The airport search / home page.                                      |

Tips:

- Any resolution is fine. If an image renders too large in the README, the
  reference can be swapped to an HTML tag with a fixed width, e.g.
  `<img src="screenshots/scope.png" width="800" alt="FreqScope radar scope">`.
- Keep file sizes reasonable (PNG is fine; compress if a shot is very large).
  The hero `scope.png` is resized to ~1600px wide for README display.
